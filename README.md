# UTS Pemrograman Web 2 - Online Shop

## Deskripsi Project
Aplikasi Online Shop sederhana (Frontend) yang dibangun untuk memenuhi tugas UTS Pemrograman Web 2. Aplikasi ini menggunakan LocalStorage sebagai simulasi database untuk manajemen user session dan keranjang belanja, serta JSON untuk memuat data katalog produk.

## Fitur Utama
1. Menampilkan daftar produk dari file JSON dummy.
2. Keranjang belanja (Cart) yang menyimpan data ke LocalStorage.
3. Form Checkout untuk memproses pesanan dan menghasilkan ID Transaksi secara otomatis.
4. Tampilan responsif menggunakan Tailwind CSS.
